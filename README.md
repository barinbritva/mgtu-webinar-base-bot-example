# Launch

* Open [BotFather](https://t.me/BotFather) and register your bot
* Put given bot token to `Telegraf` constructor in `src/index.js`
* Run `npm install`
* Run `npm start`

# Usage

Try next commands:

* `/showrates`
* `/convert`
* `/convert 5`
* `/convert 5 eur`
* `/convert 5 usd`

Try send any sticker or send text `hi` or `привет`.
